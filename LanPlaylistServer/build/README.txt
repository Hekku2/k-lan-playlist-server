=====================================
Kokkeli lan playlist server lanplayer
=====================================

Setup:
1. Change vlc-folder from settings/publish.dat to correct one
2. Default uri (http://localhost:9998/) can be changed by changing ServerUri and Port-settings from publish.dat
3. Default login is
    username: admin
    password:kokkeli

Usage:
server.bat starts server
ripper.bat starts ripper
